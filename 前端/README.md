# 就业技能需求可视化前端

该目录已从静态 HTML/JS 原型迁移为需求文档中指定的 `Vue 3 + ECharts` 前端。应用优先连接后端 `http://127.0.0.1:8000`，接口不可用时会自动切换到标注为“样例数据模式”的演示数据。

## 技术栈

- Vue 3
- Vite
- Apache ECharts 6

## 本地启动

```powershell
npm install
npm run dev
```

访问：

```text
http://127.0.0.1:5173
```

## 构建

```powershell
npm run build
```

## 当前页面

- 总览：核心指标、岗位技能图谱、城市分布、热门技能和趋势。
- 岗位查询：岗位记录筛选、分页、详情和岗位局部图谱。
- 技能图谱：城市、岗位、技能、专业综合关系图。
- 城市对比：岗位规模、薪资、技能热力和岗位结构。
- 趋势预测：技能月度需求序列和未来 3 个月预测展示。
- 数据质量：来源分布、字段质量和质量雷达。
